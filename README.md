## UnicornTranscoder

This software is a part of __PlexCloud@Home__ project, it's the transcoder manager.

## Dependencies
* RedisCache
* *UnicornLoadBalancer*
* *UnicornTranscoder*

## Installation
* Clone this repository
* Run `npm i`
* Setup your environnement variables, read `utils/config.js` for more informations
* Run `npm start`
